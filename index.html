📦 RouteMaster Order Picker

Warehouse pathfinding powered by Breadth-First Search

ℹ️ How to use
  • Option A — Upload a JSON file containing GRID and START keys, then click Load.
  • Option B — Paste or edit the JSON directly in the textarea below.
  • Option C — Click cells on the grid to dynamically place walls and targets!
  • GRID / grid — 2-D array where 0 = walkable, 1 = obstacle, 2 = target item.
  • START / start — [row, col] of the picker's starting position.
  • Mark one or more cells in the grid with 2 — those are the targets. The picker visits all of them via greedy nearest-neighbour BFS.
  • Click Find Route to run BFS and view the result.

JSON must contain GRID (with one or more cells set to 2 as targets) and START keys.
